Bot管理パネル

導入サーバーの管理とBotの操作

概要

Bot管理パネルは、Botが導入されているサーバーの一覧表示や、 Botをサーバーから退出させる操作を行うことができる管理用パネルです。 Botオーナーのみが使用できます。

使い方

/管理パネル コマンドを実行すると、管理パネルが表示されます。

サーバー一覧

導入サーバーがページ送りで一覧表示されます。 サーバーを選択すると、そのサーバーの管理画面に移動します。

サーバー管理画面

管理画面では以下の操作が行えます。

操作 説明
サーバー情報の表示 サーバーIDなどの基本情報を表示
Bot退出 Bot をサーバーから退出させる

⚠ 注意: Bot管理パネルはオーナーのみが使用可能です。Botをサーバーから退出させると、そのサーバーでのすべての機能が停止します。

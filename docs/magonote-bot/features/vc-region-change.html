定期VCリージョン変更

VCのリージョンをスケジュールに従って自動変更

概要

定期VCリージョン変更機能は、指定した VC のリージョン（地域）を 一定間隔ごとに一時的に切り替え、設定した秒数後に自動的に元のリージョンに 戻すシステム機能です（ブリップ方式）。

Divo などの BGM Bot を長時間流していると音が途切れる現象が発生することがあります。 リージョンを一瞬切り替えて戻すことで、VC の経路をリフレッシュし 音声の再接続を促す効果が期待できます。

この機能はシステム Cog として動作し、Bot 起動時に自動的に読み込まれます。 データベース（MySQL）に設定を保存し、30秒間隔のループでスケジュールをチェックします。

🔐 管理者専用 この機能の設定・管理はサーバー管理者のみ実行できます。

コマンド

/vcリージョン変更設定
VCリージョン変更のスケジュール一覧を表示・管理するパネルを開きます。
管理者

使い方

1. スケジュールを作成

/vcリージョン変更設定 を実行し、「➕ 新規作成」ボタンで新しいスケジュールを作成します。 複数のスケジュールを保存でき、それぞれ独立した設定を持てます。

2. 基本設定をする

スケジュール一覧で「✏️ 編集」を押すと詳細設定パネルが開きます。 「⚙️ 基本設定」ボタンから以下の項目を設定します。

  • 設定名 — スケジュールの名前（管理用）
  • 変更前リージョン — 有効化時に最初に適用するリージョン（任意）
  • 対象カテゴリ — カテゴリ内の全 VC が対象（複数選択可）
  • 対象VC — 個別の VC を指定（複数選択可）

3. 切り替え設定をする

「🔄 切り替え設定」ボタンから、以下の項目を設定します。

💡 動作イメージ 設定時間①（分）ごとに「香港に切り替え → 設定時間②（秒）待機 → 自動で元のリージョンに戻す」を繰り返します。

設定項目

項目 説明
切り替え先 一時的に切り替えるリージョン（日本／香港／米国西部 など）
戻し先 切り替え後に戻すリージョン。
「元のリージョン」を選ぶと基本設定の「変更前リージョン」の値が使われます。
個別のリージョンを直接指定することもできます。
間隔（分） ブリップを実行する間隔（分）
保持時間（秒） 切り替え先リージョンを維持する秒数

4. 有効化

「▶️ ON/OFF」ボタンでスケジュールを有効化します。 有効にすると 30秒間隔のループが設定をチェックし、 間隔時間が経過するたびにブリップ（切り替え → 保持 → 戻し）が自動実行されます。

💡 初回有効化時 「変更前リージョン」が設定されている場合、最初にそのリージョンが適用されてからスケジュールが開始されます。

自動実行の仕組み

この機能は Bot 起動時に 30秒間隔 のループタスクを開始します。 ループの処理フローは以下の通りです。

  1. 全ギルドの全スケジュールをチェック
  2. 無効なスケジュールはスキップ
  3. 既にブリップ実行中のスケジュールはスキップ（重複防止）
  4. 最終変更時刻からの経過時間が設定間隔（分）を超えているか確認
  5. 超えている場合、非同期タスクでブリップを実行：
    1. 切り替え先リージョンを全対象 VC に適用（各 VC 間 0.3秒インターバル）
    2. 保持時間（秒）だけ待機
    3. 戻し先リージョンを全対象 VC に適用
  6. ブリップ実行中、他のスケジュールのチェックは継続
  7. 設定された通知チャンネルに切替・復帰の通知を送信（設定されている場合）

通知メッセージ例: 

🔄 VC切替 (`スケジュール1`)
リージョンを「香港」に変更しました
（10秒後に戻します）
🔄 VC復帰 (`スケジュール1`)
リージョンを「元のリージョン」に戻しました

データベース

設定は MySQL の vc_region_schedules テーブルに保存されます。 テーブル構造は以下の通りです。

カラム名 型 説明
guild_id BIGINT サーバーID（主キー）
schedule_id INT スケジュールID（主キー）
settings_json JSON 設定内容（全項目をJSONで保存）
updated_at TIMESTAMP 最終更新日時（自動更新）

settings_json には以下のようなデータが保存されます： 

{
  "name": "Divo対策",
  "enabled": true,
  "target_category_ids": [123456789],
  "target_vc_ids": [],
  "initial_region": null,
  "target_region": "hongkong",
  "return_region": "__original__",
  "interval_minutes": 60,
  "hold_seconds": 10,
  "last_changed_at": "2026-07-17T20:00:00+09:00"
}

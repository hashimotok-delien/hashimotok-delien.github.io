画像投稿スポイラー判定

指定チャンネルでのスポイラーなし画像投稿を禁止

概要

画像投稿スポイラー判定機能は、指定したチャンネルでのスポイラーのない画像投稿を禁止する機能です。 スポイラーなしで画像が投稿された場合、自動的に削除されます。

設定方法

  1. 設定パネルを開く /設定パネル を実行
  2. スポイラー必須チャンネル設定を選択 セレクトメニューから「スポイラー必須チャンネル設定」を選択
  3. 対象チャンネルを選択 スポイラーを必須にしたいチャンネルを選択（複数選択可）
  4. 保存 「戻る」→「保存」で設定を保存

⚠️ 注意: この機能が有効なチャンネルでは、スポイラー無しで画像を投稿すると自動的に削除されます。ユーザーには事前に周知しておくことを推奨します。

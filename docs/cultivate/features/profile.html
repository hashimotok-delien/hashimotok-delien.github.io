プロフィール機能

メッセージリンクからプロフィールを保存・表示

概要

プロフィール機能は、自分のプロフィールメッセージを保存し、VC 内のインチャなどで再表示できる機能です。 これにより、VC に入るたびにプロフィールを送信する手間を省けます。

プロフィールチャンネル（自動記録）

管理者が設定パネル（/設定パネル → プロフィールv1設定）から プロフィールチャンネルを登録すると、そのチャンネルに投稿されたメッセージが 自動的にプロフィールとして記録されます。

設定項目

項目 説明
プロフィールチャンネル登録 メッセージをプロフィールとして自動記録するチャンネルを設定します（複数可）
プロフィールロール紐づけ チャンネルごとにプロフィールを記録できるロールを制限できます
VCプロフィール設定 VC 入室時に自動表示するプロフィールメッセージの内容や表示方法をカスタマイズします

💡 使い方: プロフィールチャンネルを設定するだけで、メンバーが投稿したメッセージが自動的にプロフィールとして記録され、VC 入室時に表示されるようになります。 手動で保存したい場合は /プロフィール保存 コマンドも使用できます。

コマンド

プロフィール保存

/プロフィール保存 <url>

保存したいプロフィールメッセージのリンクを指定して実行します。 保存したプロフィールは VC 入室時に自動表示されます。

引数 説明
url 保存するプロフィールメッセージのリンク（同サーバーのメッセージのみ）

⚠️ 注意: 保存できるメッセージリンクは 同じサーバー のもののみです。他サーバーのメッセージリンクは使用できません。

プロフィール再表示

/プロフィール再表示

VC 内のインチャに保存されているプロフィールを全員分再表示します。 プロフィールが表示されなくなった場合などに使用します。

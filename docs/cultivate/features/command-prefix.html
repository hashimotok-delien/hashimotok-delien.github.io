接頭辞変更

コマンドの接頭辞（プレフィックス）をカスタマイズ

概要

Cultivate では、従来のプレフィックスコマンド（!?sticky など）の接頭辞（プレフィックス）部分をサーバーごとに変更できます。 デフォルトの接頭辞は !? ですが、他Botと競合する場合などに任意の文字列に変更可能です。

接頭辞の変更は設定パネルから行います。変更後はすぐに反映され、新しい接頭辞でコマンドを実行できるようになります。

💡 補足: スラッシュコマンド（/）は接頭辞の影響を受けません。接頭辞は従来のテキストコマンドのみに適用されます。

設定手順

  1. 設定パネルを開く /設定パネル を管理者権限で実行
  2. コマンド設定を選択 表示されたパネルの「⌨️ 接頭辞変更(デフォルト!?)」ボタンをクリック
  3. 新しい接頭辞を入力 表示されたモーダルに希望の接頭辞を入力（例: ? や c! など）
  4. 保存 プレビュー画面で「✅ 保存」ボタンをクリックして確定

設定画面の見方

項目 説明
現在値 現在設定されている接頭辞（初期値の場合は !? と表示）
設定ボタン 新しい接頭辞を入力するモーダルを開きます
✅ 保存ボタン 入力された接頭辞を保存します（変更がない場合は無効）
キャンセルボタン 設定をキャンセルしてパネルを閉じます

⚠️ 注意:

  • 保存しないままパネルを閉じると変更は破棄されます
  • 変更は即座に反映されます
  • デフォルトに戻したい場合は再度設定画面から !? を入力して保存してください

プロフリアクション

プロフィールに付いたリアクションを確認

概要

プロフリアクション機能は、自分のプロフィールメッセージに付いたリアクションを確認できる機能です。 誰がどのリアクションを押したかをチェックできます。

ユーザー用コマンド

リアクション確認

/プロフリアクション確認

自分のプロフィールに付いたリアクション情報を調べます。 結果は DM で送信されます。誰がリアクションしたか、どの絵文字が押されたかを確認できます。

管理者用コマンド

プロフリアクション設定

/プロフリアクション設定 <モード>
モード 説明
登録 プロフィールに使用するリアクションの絵文字を登録
確認 登録されているリアクション設定を確認
削除 登録されているリアクション設定を削除
計測 リアクションの統計情報を計測・表示

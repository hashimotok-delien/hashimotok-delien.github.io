予約投稿

メッセージを指定日時に自動投稿

💡 予約投稿とは？ Discordのメッセージリンクを登録し、指定した日時に 自動で再投稿する機能です。 お知らせやイベント告知などを事前に準備しておけます。

概要

予約投稿機能は、メッセージリンクを設定に登録し、 指定した日時になると自動的に指定チャンネルに投稿します。 投稿にはWebhookを使用することも可能で、 名前やアイコンをカスタマイズして送信できます。

バックグラウンドループが30秒ごとに動作し、 投稿時刻を過ぎた設定を自動で処理します。

機能一覧

機能 説明
メッセージリンク登録 既存のメッセージリンクから内容を取得して予約
日時指定投稿 特定の日時に自動投稿（秒単位で指定可能）
チャンネル指定 投稿先チャンネルを個別に設定
Webhook投稿 Webhookを使用して名前・アイコンをカスタマイズ
有効/無効切替 個別の予約設定を一時的に無効化可能
ステータス管理 pending / posted / cancelled の状態管理
自動ループ 30秒間隔で投稿チェック＆自動実行

設定項目

項目 説明
名前 設定の管理名
メッセージリンク 予約するメッセージのURL （https://discord.com/channels/... 形式）
投稿チャンネル メッセージを投稿する先のチャンネル
投稿日時 投稿を実行する日時（秒単位で指定）
Webhook名 Webhook使用時の投稿者名（省略可）
Webhookアイコン Webhook使用時のアイコンURL（省略可）
有効/無効 この予約設定を有効にするかどうか

使い方の流れ

  1. メッセージを準備 予約したい内容のメッセージを任意のチャンネルに投稿します。
  2. メッセージリンクをコピー 投稿したメッセージのリンクをコピーします （Shift+右クリック → 「メッセージリンクをコピー」）。
  3. /予約投稿設定 を実行 設定パネルを開き、新規作成から予約設定を登録します。 メッセージリンク、投稿先チャンネル、投稿日時を入力します。
  4. 自動投稿 指定日時になると自動で投稿されます。

コマンド一覧

/予約投稿設定
予約投稿設定パネルを開きます。 予約設定の作成・編集・削除・一覧表示が行えます。
管理者専用

注意事項

  • メッセージリンクは同じサーバー内のメッセージである必要があります
  • Botがメッセージを取得するには、元のメッセージが存在し、かつBotがそのチャンネルを閲覧できる必要があります
  • 投稿後も設定は削除されず、ステータスが posted に変わります（再度有効化して再利用可能）
  • Webhookを使用する場合、BotにWebhook管理権限が必要です
  • 投稿チャンネルはテキストチャンネルである必要があります
  • 既に投稿済みの設定を再有効化すると、同じ内容が再度投稿されます

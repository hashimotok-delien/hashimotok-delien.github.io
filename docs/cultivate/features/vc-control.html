VC制御

Bot の VC 入退室を制御し、快適な VC 環境を維持

概要

VC制御機能は、Bot の VC への入退室管理や、人数制限の自動調整を行う機能です。

Bot 接続時自動人数調整

Bot が人数制限のある VC へ入退室した際に、VC の人数制限を自動で調整します。 Bot が入室したときは制限人数を +1 し、退室したときは -1 することで、 本来のユーザー収容数を維持します。

設定方法

  1. 設定パネルを開く /設定パネル を実行
  2. 機能を ON にする 「Bot接続時自動人数調整」を ON に設定

VC入退室コマンド

/join
現在自分がいる VC に Bot を呼びます。
管理者専用
/leave
現在 Bot がいる VC から切断します。
管理者専用

空VC自動削除

Bot は VC から全員が切断された際に、自動で空の VC を削除する処理を行います。 これにより、不要な VC が残り続けることを防ぎます。

💡 ヒント: 自動人数調整機能は、裏2shot の VC やイベント用 VC など、人数制限が重要なシーンで特に役立ちます。

MBTI機能

MBTI診断や新規開拓VCでコミュニティを活性化

概要

MBTI機能は、MBTI（ Myers-Briggs Type Indicator ）に基づいた診断パネルや 新規開拓 VC パネルを設置できる機能です。コミュニティ内の交流促進に役立ちます。

コマンド

MBTI新規開拓パネル設置

/mbti新規開拓パネル設置

MBTI 新規開拓用の VC パネルを設置します。特定サーバー限定で利用できます。

MBTI診断パネル設置

/mbti診断パネル設置

MBTI 診断パネルを設置します。診断ボタン（全員 / 異性 / 同性）が付属し、 ユーザーは自分の MBTI を診断・登録できます。

💡 情報: MBTI 機能は特定のサーバー向けに開発されています。

吊下げメッセージ（スティッキー）

指定チャンネルに常時表示される固定メッセージ

概要

吊下げメッセージ機能（スティッキーメッセージ）は、指定した チャンネルにおいて設定したメッセージを最下部に常に表示し 続けることができる機能です。新しいメッセージが送信される たびに、自動的に最下部に再投稿されます。

制限事項

  • 吊下げメッセージは 1チャンネルにつき1つ まで設定可能
  • 設定は Bot の再起動後も保持されます
  • 1サーバーにつき最大100個 まで設定可能
  • Webhook を使用する場合（&name / &icon 設定時）、ボタンは使用できません

コマンド

設定（プレフィックスコマンド）

!?sticky &count=5 &time=5 &name=送信者名
&icon=https://example.com/icon.png
&content=本文 &embed_title=タイトル

削除（プレフィックスコマンド）

!?unsticky

一覧表示（スラッシュコマンド）

/sticky list

編集（スラッシュコマンド）

/sticky edit [<チャンネル>]

チャンネルを省略した場合、コマンド実行チャンネルが 対象となります。

設定オプション一覧

オプション 説明 デフォルト値

&time メッセージを送信するまでの時間（秒） 5
&name メッセージ送信者名（設定時はWebhook使用） Cultivate
&icon メッセージ送信者アイコンURL（設定時はWebhook使用） Botアイコン
&content メッセージ本文（改行や空白も可能） -
&embed_title 埋め込みメッセージのタイトル -
&embed_description 埋め込みメッセージの説明文 -
&embed_color 埋め込みメッセージのカラー （#RRGGBB 形式） #000000
&embed_footer 埋め込みメッセージのフッター文 -
&embed_thumbnail 埋め込みメッセージのサムネイルURL -
&embed_image 埋め込みメッセージの画像URL -
&embed_author 埋め込みメッセージの著者名 -
&embed_url 埋め込みメッセージのURL（タイトルクリック時） -
&embed_timestamp 埋め込みメッセージにタイムスタンプを追加 -
&silent メッセージ送信時の通知を非表示 （true/false） false
&no_mention メンション通知を送信しない （true/false） false
&no_bot Botのメッセージで更新しない （true/false） false
&profile_toc プロフィールv2の目次を自動生成 （true/false） false
&show_my_profile 自分のプロフィールを取得するボタンを表示 （true/false） false

💡 ヒント: &name または &icon を設定した場合、Webhook を使用して メッセージが送信されます。カスタムキャラクター機能と 組み合わせることで、見た目をさらにカスタマイズできます。 ただし、Webhook モードではプロフィールボタンは使用できません。

編集 UI（設定パネル）

/sticky edit を実行すると、ボタン形式の設定 パネルが表示されます。

編集画面の操作

ボタン 説明
📝 投稿条件 更新メッセージ回数・更新秒数を設定
✏️ 投稿設定 投稿者名・アイコンURL・メッセージ本文を設定
👤 投稿者設定 埋め込みタイトル・説明・色・フッター・著者名を設定
📎 埋め込みURL等 埋め込みURL・画像・サムネイル・タイムスタンプを設定
⚙️ 機能設定 通知OFF・メンションOFF・Bot反応OFF・ プロフ目次・プロフボタンのON/OFFを設定
🔄 再投稿 現在の設定でメッセージを手動で再投稿
🔁 有効/無効 吊下げメッセージのON/OFFを切り替え
🗑️ 削除 吊下げメッセージ設定を削除

機能設定の詳細

「⚙️ 機能設定」ボタンを押すと、チェックボックス形式で 以下の項目を個別にON/OFFできます：

項目 説明
通知OFF メッセージ送信時の通知を非表示にする
メンション通知OFF メッセージ内のメンション通知を抑制する
Bot反応OFF Botのメッセージによる更新を無視する
プロフ目次 プロフィールv2のメッセージ一覧リンクを 自動生成して追記する
自分のプロフ取得ボタン 各ユーザーがワンクリックで自分のプロフィールを 取得できるボタンを表示
プロフ検索ボタン 名前やIDからプロフィールを検索できるボタンを表示

プロフィールボタン

吊り下げメッセージにプロフィール関連のボタンを 追加できます。

自分のプロフィールボタン 👤

  • メッセージ下部に「自分のプロフィールを見る」 ボタンが表示されます
  • クリックするとプロフィールv2またはv1のカードが 表示されます
  • Botメッセージとして送信されるため、他人に 見られる心配はありません

プロフィール検索ボタン 🔍

  • メッセージ下部に「プロフィールを検索」 ボタンが表示されます
  • クリックするとモーダルが開き、ユーザー選択 または名前での検索が可能です
  • 部分一致検索に対応し、複数候補がある場合は 一覧から選択できます

⚠️ 注意: プロフィールボタンは Webhook を使用したメッセージでは表示されません。 Bot として直接送信されるメッセージでのみ利用可能です。

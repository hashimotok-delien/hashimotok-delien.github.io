吊下げメッセージ（スティッキー）

指定チャンネルに常時表示される固定メッセージ

概要

吊下げメッセージ機能（スティッキーメッセージ）は、指定したチャンネルにおいて設定したメッセージを最下部に常に表示し続けることができる機能です。 新しいメッセージが送信されるたびに、自動的に最下部に再投稿されます。

制限事項

  • 吊下げメッセージは 1チャンネルにつき1つ まで設定可能
  • 吊下げメッセージは Bot の再起動後も保持されます
  • 吊下げメッセージは 1サーバーにつき最大100個 まで設定可能

コマンド

設定（プレフィックスコマンド）

!?sticky &count=5 &time=5 &name=送信者名 &icon=https://example.com/icon.png &content=本文 &embed_title=タイトル
オプション 説明 デフォルト
&count メッセージを送信するまでの通常メッセージ数 5
&time メッセージを送信するまでの時間（秒） 5
&name メッセージ送信者名（設定時はWebhook使用） Cultivate
&icon メッセージ送信者アイコンURL（設定時はWebhook使用） Botアイコン
&content メッセージ本文（改行や空白も可能） -
&embed_title 埋め込みメッセージのタイトル -
&embed_description 埋め込みメッセージの説明 -
&embed_color 埋め込みメッセージのカラー（#RRGGBB） -
&embed_footer 埋め込みメッセージのフッター文 -
&embed_thumbnail 埋め込みメッセージのサムネイルURL -
&embed_image 埋め込みメッセージの画像URL -
&embed_author 埋め込みメッセージの著者名 -
&embed_url 埋め込みメッセージのURL（タイトルクリック時） -
&embed_timestamp 埋め込みメッセージにタイムスタンプを追加 -
&silent メッセージ送信時の通知を非表示（true/false） false
&no_mention メンション通知を送信しない（true/false） false
&no_bot Botのメッセージで更新しない（true/false） false

削除

!?unsticky

指定チャンネルの吊下げメッセージを削除します。

一覧表示

/sticky list

サーバー内の全てのチャンネルの吊下げメッセージ設定を一覧表示します。

編集

/sticky edit [<チャンネル>]

指定チャンネルの吊下げメッセージ設定を編集します。チャンネルを省略した場合、コマンド実行チャンネルが対象となります。

💡 ヒント: &name または &icon を設定した場合、Webhook を使用してメッセージが送信されます。カスタムキャラクター機能と組み合わせることで、見た目をさらにカスタマイズできます。

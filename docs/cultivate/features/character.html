キャラクター設定

吊り下げメッセージなどで使用するキャラクターを作成・管理

概要

キャラクター設定機能は、吊り下げメッセージ（スティッキー）などで使用する キャラクター（名前・アイコン）を作成・管理できる機能です。 キャラクターを設定することで、用途に応じた表示名やアイコンでメッセージを送信できます。

コマンド

/キャラクター設定

コマンドを実行すると、現在登録されているキャラクターの一覧が表示されます。 以下の操作が可能です。

操作 説明
➕ 新規作成 新しいキャラクターを作成します
名前 キャラクターの名前を設定（未設定時は Bot の名前）
アイコン キャラクターのアイコンを設定（未設定時は Bot のアイコン）

作成したキャラクターは、吊り下げメッセージの Webhook 送信時に 名前やアイコンとして使用できます。

💡 ヒント: キャラクターを活用することで、運営からのお知らせやイベント案内など、用途に応じた見た目でメッセージを送信できます。

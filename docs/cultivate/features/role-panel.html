役職パネル

ユーザー自身がロールを選択・取得できるパネルを設置

概要

役職パネル機能は、ユーザー自身がつけたいロールを選択して取得できる機能です。 管理者は設定次第で、ユーザーが複数のロールを取得できるようにしたり、1つのロールしか取得できないようにしたりできます。 また、剥奪専用や付与専用のパネルも作成可能です。

パネルの種類

種類 説明
付与専用 ユーザーがロールを取得できるパネル（剥奪不可）
取外専用 ユーザーがロールを剥奪できるパネル（付与不可）
許可 ユーザーが複数のロールを取得・剥奪できるパネル
禁止 ユーザーが1つのロールしか取得できないパネル（排他選択）
特殊 重複禁止で、ロールの取り外しが可能

コマンド

新規作成

/役職パネル作成 <タイトル> <種類> <ロールA> [<説明>] [<ロールB>]...
引数 説明
タイトル パネルのタイトル
種類 付与専用 / 取外専用 / 許可 / 禁止 / 特殊
ロールA, ロールB... パネルに表示するロール（最大25個）
説明 パネルの説明（省略可能）

編集

  1. パネルを選択 Bot が送信したパネルメッセージを右クリック（長押し）→「アプリ」→「役職パネル選択」を押下
  2. 編集コマンドを実行 /役職パネル編集 <種類> を実行
    種類: タイトル / 説明 / 重複許可
  3. 値を変更 タイトルと説明は入力欄から変更。重複許可はセレクトボックスから選択

💡 ヒント: 役職パネルは Bot からのメッセージとして送信されます。パネルを削除したい場合は、該当メッセージを削除してください。
